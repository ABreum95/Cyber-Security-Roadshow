## To build docker:
`sudo docker build -t tagname path/to/Dockerfile`

## To run docker:
`sudo docker run --name test -p 8080:80 -p 22:22 tagname`

## To stop docker:
`sudo docker stop test`

## To kill docker:
`sudo docker rm -f test`
